# POE2 剧情计时器项目上下文

## 当前定位

这是一个面向《流放之路 2》剧情开荒的轻量桌面计时器，产品名为“暗黑龙剧情计时器”。核心价值是：在不遮挡游戏、不影响操作的前提下，记录每个剧情阶段耗时，并用历史最快成绩帮助玩家复盘路线。

## 第一版范围

- Windows 桌面原型。
- Electron 透明置顶悬浮窗。
- 首次打开默认显示 100% 可见背景；设置中的 `透明度` 只调整背景，不影响文字和时间数字。
- 启动后默认可点击/可拖动，方便先摆放位置。
- 点击右上角锁定图标后切换为点击穿透并禁止拖动；锁定后右上角按钮仍显示，但只有独立锁按钮可点击解锁。
- 右上角控制使用图标按钮：迷你/列表/列表2、锁定、透明度、设置、最小化、关闭。
- 点击设置会单独打开设置窗口，主计时器窗口继续保留；设置窗口采用左侧图标栏 + 右侧内容区，包含关卡模板、历史记录、调参数、快捷键。
- 设置窗口从主计时器当前位置打开，默认覆盖主计时器，而不是出现在右侧。
- 计时面板和设置面板都不支持用户手动拖拽改变窗口大小。
- 迷你模式右上角第一个图标按钮按 `迷你 -> 列表模式 -> 列表模式2 -> 迷你` 循环；列表模式是在迷你模式基础上增加时间列表，列表模式2只显示关卡和当前时间。
- 列表模式的时间列表显示在四个控制按钮下方。
- 列表模式中的历史列文案使用“历史最佳时间”。
- 迷你模式使用横向布局，当前章节时间居中突出显示；`总时间` 文案在章节名下方，总时间数值在章节时间下方；暂停时只改变计时颜色，不显示暂停文字。
- 锁定后隐藏开始、下一关、重置当前关卡、重置所有关卡四个操作按钮，并自动收紧窗口高度。
- 全部关卡完成后，迷你/列表模式的章节名继续显示最后一个节点，不显示空节点或回退到第一章。
- 所有计时显示统一使用 `00:00:00` 格式。
- 历史记录中置顶展示“历史最佳记录”，由每个节点历史最快组合生成。
- 历史记录支持保存本次、删除单条记录；设置窗口中先选择关卡模板，再在该模板下选择历史记录，历史最佳记录在下拉列表中置顶并带“最佳”标识，副标题显示记录时间点；普通完整/未完成记录从 1 开始编号，列表超过 5 条后滚动显示。
- 快捷键记录：开始、暂停、继续、记录节点、重置当前关卡、迷你 / 列表 / 列表2。
- 快捷键设置采用按键捕获，不使用手动打字输入。
- 快捷键设置需要防呆校验：四个动作不能使用重复快捷键；不能留空；不能使用单独普通按键、只用 Shift 的普通组合、常见系统快捷键或编辑快捷键；新快捷键注册失败时不保存设置，并恢复旧快捷键，避免影响计时器正常使用。
- 重置当前关卡、重置所有关卡、重置关卡模板使用自定义确认弹层，不使用系统默认确认框。
- 快捷键触发重置当前关卡时直接执行，不弹确认；鼠标点击重置当前关卡/重置所有关卡按钮时需要确认。
- 默认节点：第一章、第二章、第三章、第四章、间章、异界。
- 自定义节点模板，单个模板最多 15 个节点。
- 保存关卡模板时只禁止同名模板；相同节点结构可以用不同模板名保存，历史最快仍按节点结构对比。
- 删除关卡模板，删除模板不删除历史记录。
- 历史成绩保存。
- 同节点模板下的历史最快对比。
- 记录节点后的历史对比动画固定显示 3 秒，显示在当前时间右侧的固定预留列，出现/消失时不能改变窗体宽度或主读数左右位置，绝不能遮挡或重叠主计时时间；每次记录节点都重新播放动画；文案使用自然中文结果，例如 `快 3秒`、`慢 1分12秒`，不使用 `快00:00:00` 这类机械格式，非 0 差值最少显示 1 秒。

## 暂不做

- 不做游戏内存读取。
- 不做游戏画面识别。
- 不做自动 split。
- 不做安装包和自动更新。
- 不做云同步。
