# 暗黑龙剧情计时器

一个用于《流放之路 2》剧情开荒的 Windows 透明悬浮计时器原型。

## 功能

- 默认节点：第一章、第二章、第三章、第四章、间章、异界。
- 使用暗黑龙 logo，默认名称为 `暗黑龙剧情计时器`。
- 托盘、窗口、任务栏和未来安装包/桌面快捷方式图标统一使用暗黑龙 Logo。
- 安装程序默认安装目录使用 `anheilong_timer_版本号`，例如 `anheilong_timer_0.1.0`；更换安装地址时会自动补上该目录名。
- 首次打开默认显示 100% 背景不透明度、置顶，并且先处于可点击/可拖动状态，方便调整位置。
- 设置里的 `透明度` 只调整背景不透明度，不影响文字和计时数字；迷你模式右上角也有独立透明度图标可快速调整。
- 点击设置会单独打开一个设置窗口，主计时器继续悬浮显示；设置窗口采用左侧图标栏 + 右侧内容区，包含关卡模板、历史记录、调参数、快捷键。
- 右上角是图标按钮：迷你/列表/列表2、锁定、透明度、设置、最小化、关闭。
- 锁定后主面板不可移动并且点击穿透，右上角按钮仍显示，但只有独立锁按钮可点击解锁。
- 迷你模式右上角第一个图标按钮可切换迷你 / 列表模式 / 列表模式2。
- 列表模式的关卡列表显示在四个控制按钮下方。
- 计时面板和设置面板不支持手动拖拽改变窗口大小。
- 迷你模式为横向布局，当前章节时间居中放大；`总时间` 文案在章节名下方，总时间数值在章节时间下方；暂停时只改变时间颜色，不显示暂停文字。
- 历史记录会置顶显示 `历史最佳记录`，按每个节点的历史最快组合生成。
- 点击右上角锁定图标后不再影响游戏鼠标操作；点击独立锁按钮可恢复可拖动。
- 支持快捷键开始、暂停、记录节点、重置当前关卡、切换迷你 / 列表 / 列表2。
- 快捷键触发重置当前关卡时直接执行，不弹确认；鼠标点击重置按钮时弹出确认。
- 暂停或当前章节时间为 `00:00:00` 时，可以点击 mini 面板时间手动修改当前章节耗时。
- 支持自定义剧情节点模板。
- 支持删除自定义剧情节点模板，删除模板会连带删除该模板创建的历史记录，并在确认弹层中明确提示。
- 支持本地历史成绩和同模板历史最快对比。
- 所有按钮、菜单和提示文案使用中文。
- 所有确认动作使用自定义弹窗，不使用系统默认确认框。
- 历史记录支持保存本次、删除单条记录；设置窗口中使用下拉框选择历史记录。

## 运行

```bash
npm install
npm run dev
```

## 打包

```bash
npm run dist:win
```

打包后的安装包输出在 `release/` 目录。安装版会通过相对资源路径加载前端页面，避免 `file://` 环境下出现面板不显示。

安装完成后不会自动启动程序，避免安全软件在安装器仍在写入 Electron DLL 文件时把新启动的程序误判为修改关键 DLL。当前安装包未做商业代码签名，360、Windows SmartScreen 等安全软件仍可能提示风险；正式对外分发前建议购买代码签名证书并对 exe/安装包签名。

## 默认快捷键

- `F8`：开始 / 暂停 / 继续
- `F9`：记录当前章节
- `F10`：重置当前关卡
- `F11`：迷你 / 列表 / 列表2

快捷键可在 `设置` 中修改。

## 安全边界

这个工具只做独立计时，不读取游戏内存，不注入游戏进程，不识别游戏画面，也不自动向游戏发送键鼠操作。

如果游戏使用独占全屏，部分系统上置顶悬浮窗可能无法显示；建议使用无边框窗口或窗口化全屏。
